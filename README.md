## Overview
To-do app is a to-do list app that adds items to the database and deletes the list, presenting iOS SwiftData/Core Data skills.

## Demo
![App Demo GIF](screenshots/Todo-s.gif)  


## Features
- **Core Functionality**: Adding items and deleting them(List) from the database using SwiftData/Core Data.
- **UI/UX**: Responsive design with SwiftUI for adaptive layouts across iPhone/iPad.

## Tech Stack
- **Languages/Frameworks**: Swift 6.2, SwiftUI
- **Data Management**: SwiftData (or Core Data)
- **Tools**: Xcode 16.2, Git for version control

## Installation & Setup
1. Clone the repo: `git clone https://github.com/pixieducky/swift-todo.git`
2. Open in Xcode: Navigate to the project folder and open `ToDo.xcodeproj`
3. Build & Run: Select an iOS 26 simulator/device and hit Cmd+R.

**Requirements**: macOS with Xcode 16+, iOS 26 SDK.

## Usage
- Launch the app in simulator.
- Add To-do list item in the text field and delete the list by tapping a trash symbol.
- Explore code: Check `ToDoData` for app's Model, `ContentView` for the use of Context to track app's types.
